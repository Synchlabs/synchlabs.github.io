Installation Instructions
1. Installing BetterDiscord

You can download BetterDiscord from betterdiscord.app.

If you have any questions regarding BetterDiscord, check bd.zerebos.com for user guides or join one of the two BetterDiscord support servers: BetterDiscord, BetterDiscord2.

2. Downloading the theme

Go to the Latest GitHub release and download the .css file from the assets dropdown.

3. Installing the theme

Move the theme file to your BetterDiscord theme folder:

Windows: %appdata%\BetterDiscord\themes\ Mac: ~/Library/Preferences/BetterDiscord/themes

If you have no idea how to open the according directory:

Open Discord
Open your User Settings
Go to the BetterDiscord tab
Go to the Themes tab
Click on the Open Theme Folder button
4. Enabling the theme

Make sure the theme file is placed in the correct folder and the file extension is .theme.css
(If you don't have "Show file extensions" enabled it will only show .theme!)

Head back to the themes tab and check the checkbox next to the Theme you just installed!
